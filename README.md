# 🚀 Kavin Kumar - Full Stack Developer Portfolio

A modern, cyberpunk-themed portfolio website showcasing my skills, experience, and achievements as a Full Stack Developer. Built with pure HTML, CSS, and JavaScript for optimal performance and stunning visual effects.

## ✨ Features

### 🎨 **Cyberpunk Design**
- Neon green color scheme with glowing effects
- Animated background elements and matrix-style effects
- Glitch effects and smooth transitions
- Responsive design for all devices

### 📱 **Responsive Layout**
- Mobile-first design approach
- Tablet and desktop optimizations
- Touch-friendly navigation
- Adaptive image sizing

### 🎯 **Interactive Sections**
- **About**: Animated profile with flip effect and binary overlay
- **Experience**: 3D flip cards with hover animations
- **Skills**: Collapsible categories with animated matrix background
- **Projects**: Accordion-style feature showcase
- **Achievements**: Image gallery with static layout
- **Contact**: Terminal-style interface with animated icons

## 🛠️ **Technologies Used**

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with animations and transitions
- **Icons**: Custom icon integration
- **Fonts**: Orbitron (for headings), Courier New (for terminal)
- **Responsive**: CSS Grid and Flexbox

## 🚀 **Quick Start**

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   open index.html
   # or
   python -m http.server 8000
   # then visit http://localhost:8000
   ```

3. **No build process required!**
   - Pure HTML, CSS, and JavaScript
   - No dependencies or package managers needed
   - Works directly in any modern browser

## 🔧 **Customization**

### **Colors**
Update the CSS custom properties in `style.css`:
```css
:root {
  --neon: #00ff66;
  --neon-soft: rgba(0,255,102,0.18);
  --cyan: #00ffaa;
  --light-green: #aaffdd;
}
```

### **Content**
- Update personal information in `index.html`
- Replace images in the `assets/` folder
- Modify project descriptions and achievements
- Update contact information

### **Animations**
- Adjust animation durations in CSS
- Modify keyframe animations
- Change hover effects and transitions
